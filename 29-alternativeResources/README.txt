Simple Calculator with Locale Support. Our application satisfies the following requirements:
-We include another device configuration (Locale Support) that triggers Android's runtime support to use alternative resources
-Our Simple Calculator app supports additional locales (English, French Canadian, Japanese, German)
-Our recording shows the locale we have provided additional resources for
